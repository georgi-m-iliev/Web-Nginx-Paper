Nginx може да бъде инсталиран на множество архитектури, изпълняващи различни операционни системи, включително Linux, FreeBSD, Solaris, macOS, Windows и други. Основният фокус на разработката остава върху Unix-базираните системи, където производителността и стабилността на Nginx са най-добри.

Под Windows

Версията на Nginx за Windows използва native Win32 API извиквания (не се използва Cygwin емулиращ слой за Unix под Windows). Поради тези ограничения, висока производителност и мащабируемост не трябва да бъдат очаквани. Поради тази и други причини, Windows версията се счита за бета версия. Към момента тази версия предоставя същите функционалности като Unix версията с няколко изключения:

  • XSLT filter – за обработка на XML чрез XSLT трансформации;
  • Image filter – за динамична обработка на изображения;
  • GeoIP модул – за геолокация по IP адрес;
  • Вграден Perl интерпретатор – за изпълнение на Perl скриптове в сървъра;
  • Макар и да стартира множество работни процеси, само един от тях извършва реална работа по обработка на заявките;
  • UDP (и следователно QUIC) функционалността не се поддържа.

    може да бъде компилиран от изходен код. Макар и по-гъвкав, този случай може да бъде сложен за начинаещи потребители.
